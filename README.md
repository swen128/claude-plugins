# Harness

Plugins for [Claude Code](https://claude.ai/code).

## Installation

```
/plugin marketplace add swen128/claude-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [todo-enforcer](./todo-enforcer) | Blocks Claude from stopping until all TODO items are completed |
| [comment-checker](./comment-checker) | Detects comments in code and prompts Claude to justify or remove them |
